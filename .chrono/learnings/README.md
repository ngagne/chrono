# Chrono Learnings

Store reusable learnings, error notes, and feature requests here.

Chrono uses this directory for structured workflow improvement artifacts:

- `LEARNINGS.md` for durable fixes, conventions, and corrections
- `ERRORS.md` for reusable debugging notes from command or tool failures
- `FEATURE_REQUESTS.md` for gaps worth turning into future capabilities

The `chrono-self-improvement` skill reads and writes these files as part of the Chrono workflow.
